# DDJ-FLX4 で macをコントロール
## メディアキー
#### musicアプリやYouTube Music等で下記の操作ができます
| 操作            |
| --------------- |
| 音楽の再生/停止 |
| 前の曲へ        |
| 次の曲へ        |

## YouTubeショートカット
#### YouTube動画の下記操作ができます
| 操作          |
| ------------- |
| 10秒進む/戻る |
| 再生速度変更  |


## 使用方法
1. macとDDJ-FLX4を接続する
1. スクリプトを実行する
1. DDJ-FLX4のキーを押すことでmacをコントロール
